# Knowledge Base

This folder contains the city's scraped data used for RAG.

## To Populate

Run the scraper to fetch data from your target city's website:

```bash
cd scripts
python scrape_kitchener.py
```

## What's Expected

Each `.txt` file should contain plain text that can be chunked and embedded.

## Why Empty

We don't include scraped data in the public repo. It's your competitive advantage. Scrape your own city's data for your use case.
